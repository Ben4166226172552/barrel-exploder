# barrel-exploder
Actionreiches Arcade-Game aus der "Experimenten-Höhle"! Zerstöre Öl-Fässer per Tap oder Klick und knacke den Highscore. ​4 Modi: Klassisch (mit Kettenreaktion), Zielmodus, Fallende Fässer &amp; Zeitmodus. ​Feature: Multi-Touch Support für maximale Speed im Zeitmodus! ​Highscores: Speicher deine Bestleistungen.
